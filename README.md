An AI-powered DeepFake detection engine that analyzes images for manipulated content using deep learning techniques. Built with PyTorch and trained on the FaceForensics++ dataset, this model detects visual inconsistencies and deepfake artifacts with high accuracy.

🔹 Features:
✔️ DeepFake image detection using CNN-based models
✔️ Pre-trained model integration 
✔️ Face preprocessing enhanced accuracy
✔️ Explainability with heatmaps and feature visualization

🚀 Tech Stack: PyTorch, FaceForensics++, OpenCV, NumPy

 Future Scope: Expanding to DeepFake video detection and robustness against adversarial attacks.
